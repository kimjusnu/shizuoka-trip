# 시즈오카 여행 플래너

배포 주소: [https://shizuoka-trip.vercel.app/](https://shizuoka-trip.vercel.app/)

## 여행 한눈에 보기

- 일정: 2박 3일 (4/12 ~ 4/14)
- 베이스캠프: KOKO HOTEL 시즈오카
- 동선: 시즈오카 시내 → 후지산 뷰 포인트(후지노미야/타누키호수) → 시내 복귀

## Day 1

- 시즈오카 공항 도착 후 공항버스로 시내 이동
- 숙소에 짐 맡기고 점심(함바그/참치덮밥 등 유동 조정)
- 나나야, 타마루야 등 시내 산책
- 미호노 마츠바라, 마구로관, 아오바 오뎅거리

## Day 2

- 꿈의 대교, 후지 혼초 상점가
- 후지노미야역에서 버스시간표 확인 후 식사
- 타누키호수 산책
- 이온몰 쇼핑 후 시즈오카 복귀

## Day 3

- 호텔 조식
- 시즈오카 센겐 신사, 슨푸성 공원, 현청 전망대
- 점심(정식/소바 선택)
- 키르훼봉, 파르코, 짐 회수 후 공항 버스 탑승

## 메모

- 일정표에서 `이동하기` 버튼으로 바로 지도 이동 가능
- `정보 & 체크` 탭에서 준비물/짧은 일본어 문장 확인 가능
